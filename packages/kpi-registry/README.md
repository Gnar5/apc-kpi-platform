# KPI Registry (Source of Truth)

This registry defines marketing + sales KPIs with operational rules:
- grain/time basis
- territory + segment
- inclusion/exclusion rules
- attribution rules
- exception handling
- source mappings (ServiceTitan, GA4, Ads, etc.)

These files are intended to drive:
1) Documentation (human)
2) SQL generation (machine)
3) Data quality tests (machine)
